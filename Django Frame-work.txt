Software Technology:- Software Technology are non-installable software, but the software developed by using software technology are installable 
Software Technology are two types:-
1.Open Source Technology:-Source code of open source technology is available for user. user can make changes in open source technology code.
2.Closed Source Technology:-Source code of closed  source technology is not available for user . User can not modify code.
--------------------------------------------------------------------------------------------------------------
Framework:-Framework are installable software they provide us pre-designed architecture. By using frame-work you can develop software application more easily. If u develop software by using software technology then you need to write common and specific code both .And if you develop software by using framework then you need to write specific code only , framework take care of common code.

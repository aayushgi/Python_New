# **use of create command to create a new database:**



**syntax**

	create database <database\_name>;
   
&nbsp;	eg:

&nbsp;	   create database dsdb2;





**use of create command to create a table**

**syntax**

	create table <table name>

&nbsp;	(

&nbsp;	<column\_name> <datatype>

&nbsp;	<column\_name> <datatype>

&nbsp;	<column\_name> <datatype>

 

&nbsp;	);

**E.g:**

&nbsp;   create employee table:

&nbsp;	create table employee

&nbsp;	(

&nbsp;	empid int primary key,

&nbsp;	empname varchar(50),

&nbsp;	salary int

&nbsp;	);     





#### **Primary key:**

it is a field in a table used to identify each record uniquely.it is atomic and not null.

&nbsp;	



**use of insert command to insert record into table**

**syntax**

	insert into <table\_name> values(<value1>,<value2>,<value3>);

**eg**

&nbsp;insert into employee values(1001,'rohit',80000);

**insert multiple record**

 insert into employee values(1002,'mohit',55000),(1003,'rachit',60000);





**use of select command to select record from table:**

1)select all rows along with all columns.

&nbsp;	select  \* from employee;

2)select all rows along with specific columns.

&nbsp;	select empid, empname from employee;
